# 2. 同一位上的0或1可以区分成2队。

比如LeetCode260，想要寻找2个唯一的数值，可以通过a & (~a + 1)来分裂成2个队伍，假如存在这样的一位比如0001，就说明这个1这个数值在全异或计算后，这一位一定存在2个值一个是第一位是1和第一位是0的2个数值。通过这一位可以分裂成2个队伍。

```
例如：[0001 0010 0010 0101]

for循环^ = a =  0100
a' = a & (~a + 1) = 0100
for循环^(i ^ a') = {
        1.true => int X ^= i;
        2.false => int Y ^= i;
    }
这样就分离出来了只出现一次并且是2个不同的数字
```
